# 🏐 2026 第十三屆國北烤雞盃混合排球賽

國立臺北教育大學烤雞盃混合排球賽的官方賽事網站：對外公開即時比分、賽程與戰績，並提供一套完整的
後台讓賽事工作人員管理整個賽事。

## 給觀眾／參賽者（前台）

- **即時比分**：三場地同步記分板，得分即時更新、翻牌動畫，並可為喜歡的隊伍投人氣票。
- **賽程表**：依比賽日分頁的時間 × 場地時刻表。
- **對戰成績**：
  - 循環賽對戰圖 — 每組依隊數自動畫成三角形／四邊形，一眼看出誰打贏誰、各局比分。
  - 複決賽雙敗淘汰樹狀圖。
  - 小組排名（勝場、失局、得失分比即時計算）。
- **參賽名單**：各隊隊員名冊。
- **關於比賽**：賽事介紹、競賽章程、相關檔案下載。

## 給賽事工作人員（後台 `/admin`）

以 Google 帳號登入（需管理員權限）即可：

- 管理跑馬燈與公告
- 設定首頁 Hero 比賽日期、「立即報名」按鈕連結與開放狀態
- 建立參賽隊伍、以拖曳方式分組排位
- 編輯預賽／複賽賽程（新增、搬移、刪除時段與場次）
- 進入即時計分板為比賽計分（自動同步到前台即時比分與賽程成績）
- 結算循環賽名次（同分時可手動調整順序）
- 管理場地清單、競賽章程內容，以及「關於比賽」頁籤的相關檔案下載清單

## 快速開始

```bash
npm install
npm run dev      # http://localhost:5173
```

其餘（建置指令、Firebase 設定、資料結構、部署方式等技術細節）請見 `CLAUDE.md`。
